== Description ==

This plugin easily allows the management of a personal bibliography, with full support to BibTeX. In fact, you can import your bibliography from BibTeX files and export you bibliography (or a subset of it) to a new BibTeX file. 

In addition you can also add a bibliography reference by a simple form, that make this plugin suitable also for people who do not know anything about BibTeX.

== Installation ==

1. Unzip `bibtex-ng.zip` file
1. Upload `bibtex-ng` directory to the `/wp-content/plugins/` directory on your server
1. Activate the plugin through the 'Plugins' menu in WordPress
1. You are all set

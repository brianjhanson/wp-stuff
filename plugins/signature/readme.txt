== Description ==

This plugin adds a new menu item named "Your signature" beneath the "Users" section. There a user can define their own signature that be either of the following options:

1. Disabled, so no signature attached to any post
1. First name only (you need to have it filled in your user profile)
1. Last name only (you need to have it filled in your user profile)
1. Nickname only (you need to have it filled in your user profile)
1. Full name (you need to have first and last name filled in your user profile)
1. Anonymous
1. Custom (free text, with some placeholders that can be used to take your first name, last name, nickname, email address, and website from your user profile)

== Installation ==

1. Unzip `signature.zip` file
1. Upload `signature` directory to the `/wp-content/plugins/` directory on your server
1. Activate the plugin through the 'Plugins' menu in WordPress
1. You are all set
